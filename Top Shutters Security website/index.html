FAQ

What do the Grades mean?

Grade E - This is an entry level grade that focuses on security patrolling with a Grade E
Grade D - Also an entry level grade that focuses on perform security patrolling and access control with Grade D
Grade C - used for security role that requires some delegation
Grade B - used for Security Supervisor type role
Grade A - used for security management

How long does it take to get a certificete for each Grades?

It takes 1 week to complete and get certification for each Grade.

How long does it take to complete my FireArm?

FireArms differ, so there's no specific period for how long one will take to complete training

Do I have to attend classes or write tests?

For every Grade and every FireArm certificate, one must attend classes and write tests. Every student is given study material and is notified as to when and where classes will be held and also when tests will be taking place.
